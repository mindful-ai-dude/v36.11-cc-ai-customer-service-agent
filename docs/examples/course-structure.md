# Trainnect Corporate Technical Training Course Examples

## 1. Technical Migration Workshop Format

### Azure Cloud Migration for SQL Developers
```
Module 1: Migration Planning (Half Day)
├── 1.1 Assessment Phase (90 min)
│   ├── Current SQL Server Infrastructure Analysis
│   ├── Database Dependency Mapping
│   └── Migration Strategy Selection
├── 1.2 Azure Fundamentals (90 min)
│   ├── Azure SQL Options Overview
│   ├── Networking and Security Basics
│   └── Cost Analysis and Optimization
└── 1.3 Hands-on Lab (90 min)
    ├── Azure Portal Navigation
    └── Azure SQL Server Setup

Module 2: Migration Execution (Full Day)
├── 2.1 Data Migration Tools (120 min)
│   ├── Azure Database Migration Service
│   ├── Data Migration Assistant
│   └── BACPAC Import/Export
├── 2.2 Migration Process (180 min)
│   ├── Schema Migration
│   ├── Data Transfer
│   └── Performance Tuning
└── 2.3 Validation & Cutover (120 min)
    ├── Testing Procedures
    └── Go-Live Planning
```

## 4. Hybrid Training Format

### Kubernetes on Azure for Operations Teams
```
Pre-Workshop Online Modules
├── Containerization Basics (Self-paced)
├── Kubernetes Fundamentals (Self-paced)
└── Azure Kubernetes Service Intro (Self-paced)

In-Person Workshop (2 Days)
├── Day 1: Hands-on AKS
│   ├── Morning: Cluster Setup & Management
│   ├── Afternoon: Deployment Strategies
│   └── Evening: Lab Work
└── Day 2: Production Operations
    ├── Morning: Monitoring & Logging
    ├── Afternoon: Security & Compliance
    └── Evening: Team Challenge

Post-Workshop Support
├── Weekly Office Hours (1 month)
├── Implementation Support
└── Best Practices Review
```
